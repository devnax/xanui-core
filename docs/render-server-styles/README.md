# RenderServerStyles

`RenderServerStyles` replays every stylesheet generated by `oncss` during server-side rendering. Insert it into your HTML document's `<head>` to avoid flashes of unstyled content before hydration.

## Props

| Prop     | Type | Default | Description                                                                                                                             |
| -------- | ---- | ------- | --------------------------------------------------------------------------------------------------------------------------------------- |
| *(none)* | —    | —       | The component is self-contained and does not accept props. It simply iterates `CSSFactory` and renders `<style data-oncss="...">` tags. |

## Usage Example

```tsx
// app/document.tsx
import RenderServerStyles from 'xanui-core/RenderServerStyles'

export const Document = ({ children }) => (
  <html lang="en">
    <head>
      <meta charSet="utf-8" />
      <RenderServerStyles />
    </head>
    <body>{children}</body>
  </html>
)
```

Pair it with the same components/hooks on the client; the class names will match and React will not need to re-render.
